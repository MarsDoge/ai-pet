## 核心玩法规划（草案）
### 核心循环
动作 → 状态变化 → 反馈/对话 → 获得奖励 → 解锁新动作/物品。

### 阶段目标
1. **动机驱动**：基于状态给出行动建议与小目标。
2. **成长感**：等级、金币、道具积累有明确反馈。
3. **故事感**：叙事事件时间线可回看，形成“成长记录”。

### 下一阶段（Milestone G：核心玩法闭环）
- 完成“建议行动 + 动作反馈 + 叙事时间线”的闭环。
- 引入轻量“每日目标/任务”与小奖励（不改核心规则）。
- 让 AI 输出的记忆与叙事在 UI 中可见、可复盘。
